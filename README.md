# CLOTHLESS

An invisible fashion brand.

## What is this

A landing page and signup form for a clothing brand that sells no clothing. Built with HTML and CSS.

## Tech Stack

- HTML
- CSS
- Absolutely zero JavaScript
- One Google Font that probably didn't need to be a Google Font


## Features

- Responsive (mostly)
- A navbar
- A form that doesn't submit to anything
- Colors that scream
